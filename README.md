AC2RD (Android Call Recorder Replayer Dictaphone) é um software gratuito de gravação automática de chamadas telefônicas desenvolvido para o sistema operacional móvel Android, incluindo a funcionalidade de replayer e de ditafone.

## Principais Funcionalidades :

- Gravação automática de chamadas de entrada/saída do telefone
- Formatos de áudio suportados: 3GPP, MP3, AMR e WAV
- Replayer incorporado com visualizador de espectro de áudio
- ditafone
- Widget com atalhos nas funções principais

## Características avançadas :

- Lançamento automático do software quando o telefone é iniciado
- Filtragem de chamadas nos contatos do dispositivo
- Gravação na detecção de vibração do dispositivo
- Gravação na memória interna/externa
- Monitoramento de memória de armazenamento permanente
- Registros classificáveis por tipo/data
- Remoção automática de registros obsoletos (configurável de 1 dia a 1 mês)
- Possibilidade de bloquear registros importantes
- Efeitos de espectro de áudio: nenhum, linhas ou barras
- Registro de eventos (ações do usuário e notificação de software)
- Notificações de eventos: som, vibração e LED
- Vários temas visuais com cores personalizáveis
